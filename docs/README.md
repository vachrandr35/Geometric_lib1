# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ah / 2

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c

# Description and examples
**This library is made for calculation an area or perimeter of different kind of geometric shapes**
## Circle
### Area
_the function takes the radius value and returns the area_
**Example:** area(7) = 7 * 7 * π = 153,938
### Perimeter
_the function takes the radius value and returns the area_
**Example:** perimeter(7) = 2 * π * 7 = 43,982

## Rectangle
### Area
_the function takes values of adjacent sides and returns the area_
**Example:** area(12, 8) = 12 * 8 = 96
### Perimeter
_the function takes values of adjacent sides and returns the perimeter_
**Example:** perimeter(12, 8) = 12 * 8 = 20

## Square
### Area
_the function takes the side length of a square and returns the area_
**Example:** area(13) = 13 * 13 = 169
### Perimeter
_the function takes the side length of a square and returns the perimeter_
**Example:** perimeter(13) = 13 * 4 = 52 

##Triangle
### Area 
_the function takes values of height and base and returns the area_
**Example:** area(3, 4) = 12
### Perimeter
_the function takes values of all three sides of the triangle and returns the perimeter_
**Example:** perimeter(3, 4, 5) = 12
